# 属性

いわゆる5属性。土属性は物理にしてある。

|属性名|性質|スキル例|
|:-:|:-:|:--|
|**火**|熱乾|火球、摩天楼、導きの灯火|
|**水**|冷湿|水の槍、海嘯、霧の警句|
|**風**|熱湿|風の矢、秋風の弔い、大いなる共鳴|
|**物理**|冷乾|斬撃、屍鬼呼び、王の庭、広宵大暮|
|**星幽**|不変|久遠の光、外なるものの招来|

※星幽は4属性とは別に乗算され、実体には0.5倍の、幽体には2倍のダメージを与える。

## 実体と幽体

スキルは固有の幽契値を持ち、幽契値が4になると幽体化する。
幽体化時、スキルは星幽属性を帯び、使用するたびにMPではなく幽契を消費する。

- **実体**：基準状態。
- **幽体**：受ける無属性を半減させるが、星幽が付与されたスキルからは倍のダメージを受ける。

### スキルの幽化

幽体のとき通常スキルは*幽化*し星幽属性を帯びるようになる。

|スキル例|通常効果|幽化スキル|幽化効果|
|:--|:--|:--|:--|
|火球|威力80|霊炎球|威力80。 <br/>追撃：威力80、AoE。|
|霧の警句|相手のバフを消去。 <br/>バフが無ければ暗闇を付与。|黒霧の嘆声|相手のバフを消去。 <br/>バフが無ければ絶望を付与。|
